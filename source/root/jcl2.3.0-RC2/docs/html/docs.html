jpos.config/loader (JCL) - Docs

Documentation about the JCL comprises this set of HTML readme pages, a white paper, the JavaDOC generated docs (from the sources) and of course the JCL source files themselves.

HTML Readme

This set of HTML files.  All you need to know about the JCL and simple reference implementation.

White paper - PDF format

The JCL white paper describes the architecture/design of the JCL as well as giving some rational for its existence and future directions.

NOTE: the JCL White Paper is out of date.  It needs to account for changes in 1.2 and 2.x releases.  The dW jposloader bug #2844 was open to address this.

JavaDOC

All the JCL sources are fully documented with JavaDOC comments.  Use this to quickly browse and understand the API.

Source Files

As a last resort: "Use the source Luke!" :-)

The JCL is open source software, use the source, send feedback, fix bugs and add features (send them back).

Let's all improve the JCL in the spirit of OSS for the good of the JavaPOS community. You will be credited for any improvements you make...
